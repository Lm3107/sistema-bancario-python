# Sistema Bancário em Python

Projeto desenvolvido em Python com o objetivo de simular
operações básicas de um sistema bancário.

Este projeto faz parte do meu desenvolvimento acadêmico
em Ciência da Computação e da minha transição para a área
de programação e banco de dados.

## Funcionalidades
- Criar conta
- Consultar saldo
- Realizar depósitos
- Realizar saques
- Extrato bancário

## Tecnologias
- Python 3

## Objetivo
Praticar lógica de programação, estrutura de dados
e conceitos utilizados em sistemas financeiros.
